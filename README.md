一款基于[饭饭's Homepage 2.0](https://github.com/noisky/Homepage)制作的主題

## 安装


在博客根目录中安装

```powershell
git clone -b master https://github.com/NatsunoZero/hexo-theme-navi.git themes/navi
```

## 应用主题
修改hexo配置文件`_config.yml`，把主題改為`navi`

```
theme: navi
```
